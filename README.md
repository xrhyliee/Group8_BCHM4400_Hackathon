# 2. Hypothesis
Point mutation is a genetic mutation where a single pair of DNA or RNA has been changed, inserted, or deleted. We hypothesize for sickle cell anemia in the Hemoglobin-beta locus (HBB) that the DNA has undergone point mutation. We suspect that this is happening at the sixth codon position, A - T, causing this specific codon to code for valine instead of the expected glutamine, which the non-mutated Hemoglobin-beta locus should code for.
